## 🐛 已知问题

- macOS 上可能需要手动授权网络访问权限
- Windows 需要启用 NFS 客户端功能
- Linux 某些发行版需要安装 nfs-common

4. **向后兼容**: 配置文件格式与原 shell 脚本完全兼容
